EnumerableObject

EnumerableObject: object | Map<string, T> | [string, T][]
  • Defined in interfaces.ts:27

The initial state can be provided as either a plain JavaScript object, a Map of string keys and values, or an array of arrays of string keys and values to generate the initial state from.

Here's an example of a plain object:

{ counter: 0, updatedAt: Date.now() }

Here's a map of key, value pairs:

new Map([
  ['counter', 0],
  ['updatedAt': Date.now()]
])

Here's a doubly-nested array of key, value pairs:

[
  ['counter', 0],
  ['updatedAt', Date.now()]
]
